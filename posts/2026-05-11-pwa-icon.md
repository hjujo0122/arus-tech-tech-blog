## PWAとは

**PWA（Progressive Web App）** というものがあります。普通のWebサイトに少し設定を追加することで、スマホのアプリっぽく動かせる仕組みです。

具体的にできることは：

- ホーム画面にアイコンを追加（アプリっぽく見える）
- 全画面表示（ブラウザのURLバーが消える）
- オフラインでも動く（キャッシュを使う）

今回は「ホーム画面のアイコン」があったらうれしいなと思い、設定しました。

## Next.jsの設定

Next.jsのApp Routerには、ファイルを置くだけでアイコンやmanifestを自動配信してくれる仕組みがあります。

**アイコンファイル：**

`app/` フォルダに置くだけです。

| ファイル | 用途 |
|---------|------|
| `app/icon.png` | ブラウザのfavicon・一般的なアプリアイコン |
| `app/apple-icon.png` | iOSのホーム画面アイコン（`apple-touch-icon`） |

**manifest の設定：**

PWAとして認識させるには `manifest.json`（アプリ名・アイコン・表示方法などを書いたファイル）が必要です。Next.jsでは `app/manifest.ts` を作ると自動で配信してくれます。

```ts
import type { MetadataRoute } from "next";

export const dynamic = "force-static"; // 静的エクスポートに必要

export default function manifest(): MetadataRoute.Manifest {
  return {
    name: "あるすのてくてくブログ",
    short_name: "あるすブログ",
    description: "社内SEあるすの日記がわりブログ。",
    start_url: "/",
    display: "standalone", // ブラウザUIを非表示にしてアプリっぽく表示
    background_color: "#ffffff",
    theme_color: "#ffffff",
    icons: [
      {
        src: "/icon.png",
        sizes: "512x512",
        type: "image/png",
      },
    ],
  };
}
```

`display: "standalone"` にするとホーム画面から開いたときにURLバーが消えてアプリっぽくなります。

## 結果

iPhoneのchromeなどで `https://arus-tech-tech.web.app` を開いて「ホーム画面に追加」を選ぶと、知佳がアイコンになります。アプリっぽい見た目でちょっとうれしい。
