ブログを作ったはいいものの、SEO的にどうなの？という話になったので対応しました。

## やったこと一覧

- OGタグ / Twitter Cardの設定
- `sitemap.xml` の自動生成
- `robots.txt` の追加
- トップページの `description` 改善

## OGタグって何？

TwitterやDiscordでURLを貼ったとき、タイトル・説明・画像がいい感じに表示されるやつです。あれが出るかどうかはOGタグ次第です。

設定前はURLをそのまま貼っても何も展開されませんでした。設定後は知佳ちゃんの画像つきでプレビューが出るようになりました。やったね。

Next.jsは `metadata` オブジェクトに `openGraph` と `twitter` を書くだけで対応できます。

```ts
export const metadata: Metadata = {
  openGraph: {
    type: "website",
    title: "あるすのてくてくブログ",
    description: "社内SEあるすの日記がわりブログ。",
    images: [{ url: "https://arus-tech-tech.web.app/chika-header.png" }],
  },
  twitter: {
    card: "summary_large_image",
  },
};
```

記事ページは個別に上書きして、記事タイトルと概要が出るようにしています。

## sitemap.xml と robots.txt

Googleに「このブログにはこんなページがありますよ」と伝えるためのファイルです。

Next.jsの `app/sitemap.ts` と `app/robots.ts` に関数を書くだけで `/sitemap.xml` と `/robots.txt` が自動生成されます。便利すぎる。

sitemap.xmlにはトップ・Aboutページ・全記事のURLが含まれており、記事を追加するたびに自動で更新されます。

## description の改善

設定前は `"あるすのてくてくブログ"` という、名前をそのまま貼っただけの説明文でした。Googleの検索結果に出るスニペットにそのまま使われるので、もうちょっとましな文章にしました。

```
社内SEあるすの日記がわりブログ。技術・日常・美少女オタク話など。
```

正直なところ「美少女オタク話など」の部分がちゃんとSEOに効くのかは謎ですが、正直に書きました。
